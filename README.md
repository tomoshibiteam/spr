# TOMOSHIBI - ソーシャル・インパクト・エンターテインメント

## プロジェクト概要

**TOMOSHIBI**は、謎解きイベントとWebアプリを融合させ、社会課題への取り組みをエンターテインメント化する「ソーシャル・インパクト・エンターテインメント」プロジェクトです。

ユーザーは調査組織「SPR探偵事務所」の探偵となり、社会課題をテーマにした事件（依頼）を解決していきます。

## デザインコンセプト

### 世界観
- ゲーム『レイトン教授』シリーズのような、温和で知的、かつ少しレトロな雰囲気
- 古い探偵事務所をイメージした、アンティークで温かみのあるデザイン

### カラースキーム
- **ベース**: 温かみのある明るいブラウン (#8B6F47)
- **背景**: クリーム色 (#F5F1E8)
- **アクセント**: 深緑 (#2C4A3E)
- **ハイライト**: ゴールド・真鍮色 (#D4AF37, #B5A642)

### デザイン要素
- **テクスチャ**: アンティークな羊皮紙や革製品のような手触り感
- **フォント**: 手書き風フォントとクラシックなセリフ体の組み合わせ
- **UIパーツ**: 古い探偵道具（虫眼鏡、万年筆、コンパスなど）をモチーフ

## ページ構成

### 1. ランディングページ（index.html）

**目的**: プロジェクトの紹介と探偵登録への導線

**主要セクション**:
- **ファーストビュー**: キャッチコピーとビジュアル
- **TOMOSHIBIとは**: 世界観とコンセプトの説明
- **解決する課題**: 社会課題の提示
- **仕組み**: 日常と非日常の循環システム
- **次なる事件**: 今後のイベント予定
- **CTA**: 探偵登録への誘導

**キャッチコピー例**:
> 君の日常が、世界を救う物語になる。

### 2. マイページ - 探偵手帳（mypage.html）

**コンセプト**: 革張りの手帳風デザイン

**主要機能**:
- **探偵プロフィール**: ユーザー名、ランク、称号の表示
- **ステータス**: 貢献量の可視化（ゴミ拾い量など）
- **所持ツール**: レベルアップ可能なツール一覧
- **スキル**: イベントで役立つスキル表示
- **獲得バッジ**: 実績バッジのコレクション
- **最近の活動**: 活動履歴のタイムライン

### 3. 活動記録ページ - 調査報告（report.html）

**コンセプト**: 日常の貢献活動を簡単に報告

**主要機能**:
- **活動内容の選択**: ゴミ拾い、エコ活動、文化イベント参加など
- **活動量の入力**: 重量、時間、回数などの記録
- **詳細入力**: フリーテキストで活動の詳細を記述
- **写真添付**: 活動の証拠写真のアップロード
- **フィードバック**: 報告後の経験値とレベルアップの演出

**報告のコツ**:
- 具体的に書くことで多くの経験値を獲得
- 写真を添付するとボーナス経験値
- 継続報告で連続ボーナス

### 4. イベント一覧ページ - 依頼リスト（missions.html）

**コンセプト**: 探偵事務所の掲示板に貼られた依頼書

**主要機能**:
- **タブ切り替え**: 新着依頼 / 進行中 / 解決済み
- **依頼カード**: 事件名、開催地、難易度、あらすじ
- **進捗管理**: 進行中の事件の進捗状況表示
- **解決履歴**: 過去に解決した事件の記録

**依頼例**:
- **蒼い宝石失踪事件** (海洋ゴミ問題)
- **幻のレシピを探せ** (伝統文化の継承)
- **消えた祭りの音色** (地域活性化)

### 5. 全国支部マップ（map.html）

**コンセプト**: 古地図風のインタラクティブマップ

**主要機能**:
- **日本地図**: アンティーク調の地図デザイン
- **支部アイコン**: 各都道府県の支部表示
- **フィルター機能**: イベント開催中、探偵募集中などで絞り込み
- **活動統計**: 支部ごとの貢献量、解決事件数の表示
- **全国統計**: 全体の活動状況の可視化

## 使い方

### ブラウザで開く

1. `index.html` をブラウザで開いてランディングページを表示
2. ナビゲーションから各ページに移動
3. モックアップのため、一部機能は実装されていません（モーダル表示など）

### 対応ブラウザ

- Google Chrome（推奨）
- Firefox
- Safari
- Edge

### レスポンシブ対応

- デスクトップ: 1200px以上
- タブレット: 768px - 1199px
- スマートフォン: 767px以下

## デザインの特徴

### UIパーツ

1. **ボタン**
   - プライマリ: ゴールドのグラデーション
   - セカンダリ: クリーム色に茶色の枠線
   - ホバー時に浮き上がるエフェクト

2. **カード**
   - 羊皮紙風のテクスチャ
   - 茶色の枠線
   - ドロップシャドウで立体感

3. **アイコン**
   - 絵文字を活用した親しみやすいデザイン
   - 探偵道具モチーフ（🔍、✒️、🧭など）

4. **フォント**
   - Noto Serif JP: メインフォント
   - Sawarabi Mincho: サブフォント

### アニメーション

- フェードイン効果（ファーストビュー）
- ホバーアニメーション（ボタン、カード）
- プログレスバーの滑らかな遷移
- 回転アニメーション（循環図の矢印）
